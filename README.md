# InkBunny 작품 다운로더

## 요구사항 및 설치방법

1. Python 3.11 이상 권장
```bash
# Python 다운로드 및 설치: https://www.python.org/downloads/
python --version  # 버전 확인
```

2. 필요한 라이브러리 설치
```bash
pip install aiohttp
```

3. 프로젝트 복제
```bash
git clone https://github.com/haram03/inkbunny-downloader.git
cd inkbunny-downloader
```

4. 실행 방법
```bash
python main.py
```
